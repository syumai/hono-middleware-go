# basic-auth-middleware-go

* This is an example of Hono middleware implemented in Go.
  - a middleware of Basic-Auth .

## Demo

* https://hono-middleware-go-basic-auth.syumai.workers.dev/
* Try:
  - userName: `user`
  - password: `password`

## Development

### Requirements

This project requires `go` to be installed globally.

### Commands

```
npm run dev        # run dev server
npm run build-wasm # build Go Wasm binary
npm run deploy     # deploy worker
```
